Final Fantasy VII (Original) — Chapter 10: Grasslands & Chocobo Ranch

Step-by-step main progression + optional/missable checklist.

Map Optional reference
If this image is missing, upload it to assets/ffvii/maps/grasslands-chocobo-ranch.png.

Step-by-step

1. From Kalm, travel east/southeast across the Grasslands toward the large marsh area.
2. Before entering the marsh, visit Chocobo Farm/Ranch and speak with the ranchers.
3. Buy Chocobo Lure (and greens if desired).
4. Return to the grasslands and enter battles on the chocobo tracks to catch a chocobo.
5. Ride the chocobo across the marsh to safely reach the cave on the far side.
6. Enter the cave to continue the story route.
Optional / Missable checklist (Chapter 10)
This chapter is mostly setup for future 100% completion. Doing this now prevents later backtracking and keeps the route consistent.
Chocobo Ranch (recommended for 100%)
• Obtain Chocobo Lure so you can reliably catch chocobos.
• Learn how to use greens to keep enemies from attacking the chocobo (optional).
Marsh crossing
• Crossing on foot is possible but can cause delays; using a chocobo is the consistent route for a 100% guide.
Notes:
• Enemy Skill learning is not required in this chapter.
• If you skip the ranch now, you can return later, but most 100% routes do this immediately after Kalm.
